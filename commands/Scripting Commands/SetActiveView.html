SetActiveView Command

SetActiveView( <View> )

Makes given View active.

  • 1 or "G" for Graphics View

  • 2 or "D" for Graphics View 2

  • -1 or "T" for 3D Graphics View

  • "A" for Algebra View

  • "S" for Spreadsheet View

  • "C" for CAS View

  • See also ZoomIn, ZoomOut, Pan, SetPerspective commands.
