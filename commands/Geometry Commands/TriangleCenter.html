TriangleCenter Command

This command differs among variants of English:

  • TriangleCenter (US)

  • TriangleCentre (UK + Aus)
TriangleCenter( <Point>, <Point>, <Point>, <Number> )

gives n-th triangle center of triangle ABC. Works for n < 3054.

Let A = (1, -2), B = (6, 1) and C = (4, 3). TriangleCenter(A, B, C, 2) yields the centroid D = (3.67, 0.67) of the triangle ABC.
