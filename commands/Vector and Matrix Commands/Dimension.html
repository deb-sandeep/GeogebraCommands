Dimension Command

Dimension( <Object> )

Gives the dimension of a vector or a matrix.

Dimension({1, 2, 0, -4, 3}) yields 5.

Dimension({{1, 2}, {3, 4}, {5, 6}}) yields {3, 2}.

CAS Syntax

Dimension( <Object> )

Gives the dimension of a vector or matrix.

Dimension({1, 2, 0, -4, 3}) yields 5.

Dimension({{a, b}, {c, d}, {e, f}}) yields {3, 2}.
