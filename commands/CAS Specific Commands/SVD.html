SVD Command

SVD( <Matrix> )

Returns the Singular Value Decomposition of the matrix (as a list of 3 matrices).

SVD({{3, 1, 1}, {-1, 3, 1}}) yields a list containing \( \left(\begin{array}{rr}-0.71&0.71\\0.71&0.71\\\end{array}\right) \), \( \left(\begin{array}{rr}3.16&0\\0&3.46\\\end{array}\right)\), \(\left(\begin{array}{rr}-0.89&0.41\\0.45&0.82\\0&0.41\\\end{array}\right)\).

  • This command is also supported in the CAS View. The numbers in the answer may vary in order between the Algebra View and CAS View.
