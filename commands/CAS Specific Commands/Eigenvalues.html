Eigenvalues Command

CAS Syntax

Eigenvalues( <Matrix> )

Finds the eigenvalues of the given matrix.

Eigenvalues({{1, 2}, {3, 4}}) yields \( \left\{ \frac{\sqrt{33} + 5}{2}, \frac{-\sqrt{33} + 5}{2} \right\} \)
