PrimeFactors Command

PrimeFactors( <Number> )

Returns the list of primes whose product is equal to the given number.

  • PrimeFactors(1024) yields {2, 2, 2, 2, 2, 2, 2, 2, 2, 2}.

  • PrimeFactors(42) yields {2, 3, 7}.
