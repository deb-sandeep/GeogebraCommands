CompleteSquare Command

CompleteSquare( <Quadratic Function> )

Returns the quadratic function in the form: \(a (x - h)^2 + k\).

CompleteSquare(x^2 - 4x + 7) yields 1 (x - 2)2 + 3.

CAS Syntax
