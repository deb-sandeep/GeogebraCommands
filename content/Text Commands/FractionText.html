FractionText Command

FractionText( <Number> )

Creates and shows in the Graphics View a LaTeX text containing the fraction form of the given number.

Given line a: y = 1.5 x + 2, FractionText(Slope(a)) creates the LaTeX text \(\frac{3}{2}\).

FractionText( <Point> )

Creates and shows in the Graphics View a LaTeX text containing the fraction form of the coordinates of the given point.

Given point A=(1.33,0.8), FractionText(A) creates the LaTeX text \( \left( \frac{133}{100} ,\frac{4}{5} \right) \).

FractionText( <Number>, <Boolean Single fraction>)

Creates and shows in the Graphics View a LaTeX text containing the fraction form of the given number. The boolean sets the position of a possible negative sign of the fraction. If true, the minus sign will be displayed in the numerator, if false the minus sign will be displayed in front of the fraction.

Given the number n = -0.8 , then

  • FractionText(n, true) creates the LaTeX text \(\frac{- 4}{5}\).

  • FractionText(n, false) creates the LaTeX text \(-\frac{4}{5}\).

See also SurdText command.
