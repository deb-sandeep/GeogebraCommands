Expand Command

Expand( <Expression> )

Expands the expression.

Expand((2 x - 1)^2 + 2 x + 3) yields \(4 x^2 - 2 x + 4\).

This command needs to load the Computer Algebra System, so can be slow on some computers. Try using the Polynomial Command instead.
